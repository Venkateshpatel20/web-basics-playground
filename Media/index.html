HTML Media Attributes & CSS Media Queries

What is the media attribute in <link> and <style>?

The media attribute specifies for which device or media type the linked or embedded CSS should be applied. It can limit the stylesheets or styles to specific screen sizes, resolutions, printers, speakers, etc.

Examples of media attribute values:

  • screen: For computer screens, tablets, smartphones
  • print: For print preview and printers
  • speech: For screen readers
  • Media queries (e.g., max-width: 768px) to target devices with certain screen widths

Example usage in HTML:

<link rel="stylesheet" href="style.css" media="screen and (max-width: 768px)">

<style media="print">
  body { font-size: 12pt; }
</style>
